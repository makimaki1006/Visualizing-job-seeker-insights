## 1. システム概要

### プロダクト名
**MapComplete ダッシュボード** （医療・介護人材分析ツール）

### キャッチコピー
**「採用したい地域に、どんな求職者が、何人いるか」が一目で分かる**

### 一言説明
ジョブメドレーの求職者データを可視化し、データに基づいた戦略的採用活動を実現するダッシュボードアプリケーション。

---

## 2. このシステムができること

### 2.1 市場概況の把握
- 選択した地域の求職者総数を即座に表示
- 年齢分布（棒グラフ）
- 性別・雇用形態分布（円グラフ）
- 都道府県・市区町村TOP10ランキング

### 2.2 人材属性分析
- 資格保有状況のランキング
- 年齢×性別クロス分析
- ペルソナ分析（年齢×性別の組み合わせ）
- 定着率分析

### 2.3 地域・移動パターン分析
- 移動パターン円グラフ（地元志向/近隣移動/広域移動）
- 居住地→希望勤務地のフロー分析
- ペルソナシェア（地域内人材構成比）

### 2.4 需給バランス分析
- 需要・供給の比較
- 需給ギャップの可視化
- **レアリティ分析（人材組み合わせ検索）**
  - 年齢・性別・資格の組み合わせで検索
  - 該当者数と構成比をリアルタイム表示

### 2.5 雇用形態分析
- 雇用形態×移動パターンのヒートマップ
- 正職員/パート/派遣など詳細分布

### 2.6 地図機能
- 求人地図（Google Apps Script連携）
- 人材地図（Leaflet統合、開発中）

## 3. システムの効果・役立つ理由

### 3.1 採用担当者の悩みを解決

| 従来の悩み | このツールで解決 |
|-----------|-----------------|
| 「うちの地域に看護師はいるの？」 | **数値で把握可能** |
| 「どの資格を持った人が多い？」 | **ランキングで可視化** |
| 「隣の市から通勤してくれる人は？」 | **移動パターン分析** |
| 「競合と比べて採用しやすい？」 | **需給バランスで判定** |
| 「採用要件が厳しすぎる？」 | **レアリティ分析で検証** |

### 3.2 期待される導入効果

#### 採用コスト削減
- 無駄な求人出稿の削減（人材がいない地域を事前に回避）
- ターゲット精度向上（実在する人材層に絞った訴求）
- 採用期間短縮（データに基づく迅速な意思決定）

#### 採用成功率向上
- 競合分析（需給バランスで採用難易度を事前把握）
- 条件最適化（市場に合った給与・条件設定）
- エリア戦略（移動パターンを考慮した広域採用）

#### 経営判断の高度化
- 新規拠点開設判断（人材確保可能性を事前検証）
- 事業計画策定（採用計画の数値的根拠）
- 取締役会報告（データに基づく客観的説明）

## 7. デモ操作フロー（撮影用）

### シーン1: ログイン
1. ブラウザでURL入力
2. ログイン画面表示
3. メールアドレス・パスワード入力
4. ログインボタンクリック

### シーン2: 市場概況
1. ダッシュボード表示（デフォルト: 全国）
2. 都道府県プルダウンで「東京都」選択
3. KPIカード更新を見せる
4. 年齢分布グラフを指し示す
5. 性別・雇用形態円グラフを指し示す
6. TOP10ランキングをスクロール

### シーン3: 人材属性
1. 「人材属性」タブクリック
2. 資格ランキング表示
3. ペルソナ分析（50代×女性が最多）
4. 年齢×性別マトリックス

### シーン4: 地域・移動パターン
1. 「地域・移動パターン」タブクリック
2. 移動パターン円グラフ
3. フロー分析（居住地→希望勤務地）

### シーン5: レアリティ分析
1. 「需給バランス」タブクリック
2. 「人材組み合わせ分析」セクションへスクロール
3. 条件選択：年代「30代」、性別「男性」
4. 資格選択：「介護福祉士」にチェック
5. 検索結果：「該当45人」
6. 条件変更：年代「40代」を追加
7. 結果変化を見せる
