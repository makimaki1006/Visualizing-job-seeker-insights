# 動画用スクリプト（シンプル版）

**医療・介護人材採用を、データで変える**

---

## このシステムができること（6つの主要機能）

| # | 機能 | 内容 |
|---|------|------|
| 1 | **市場概況** | 求職者数、年齢分布、性別・雇用形態の可視化 |
| 2 | **人材属性** | 資格ランキング、ペルソナ分析、年齢×性別クロス |
| 3 | **移動パターン** | 地元志向/近隣移動/広域移動の割合、フロー分析 |
| 4 | **レアリティ分析** | 年齢・性別・資格の組み合わせで人材検索 |
| 5 | **需給バランス** | 需要と供給のギャップ可視化 |
| 6 | **地図機能** | 求人地図、人材地図 |

---

## 効果・役立つ理由

| 従来の悩み | このツールで |
|-----------|-------------|
| 「うちの地域に人材はいる？」 | → **数値で把握** |
| 「どの資格が多い？」 | → **ランキング表示** |
| 「隣の市から来てくれる？」 | → **移動パターン分析** |
| 「採用条件が厳しすぎる？」 | → **レアリティ分析で検証** |

---

## ナレーション例（90秒版）

```
【オープニング 0:00-0:15】
「採用したい地域に、どんな求職者が、何人いるか」
それが一目で分かるツール、MapCompleteダッシュボードです。

【機能紹介 0:15-0:45】
このツールには6つの主要機能があります。

1つ目、市場概況。地域の求職者数や年齢分布を即座に把握できます。
2つ目、人材属性。資格ランキングやペルソナ分析で、どんな人材がいるか分かります。
3つ目、移動パターン。隣の市から来てくれる人がどれくらいいるか分析できます。
4つ目、レアリティ分析。「30代男性で介護福祉士」のような条件で検索し、該当者数を確認できます。
5つ目、需給バランス。採用しやすさを数値で判定できます。
6つ目、地図機能。人材の分布を地図で可視化できます。

【効果 0:45-1:15】
このツールで解決できる悩みは4つあります。

「うちの地域に人材はいる？」→ 数値で把握できます。
「どの資格が多い？」→ ランキングで可視化されます。
「隣の市から来てくれる？」→ 移動パターン分析で分かります。
「採用条件が厳しすぎる？」→ レアリティ分析で検証できます。

【クロージング 1:15-1:30】
医療・介護人材の採用を、データで変える。
MapCompleteダッシュボードでした。
```

---

## 画面撮影ポイント

### 必須シーン（3つ）
1. **市場概況** → 「東京都: 132,690人」のKPI表示
2. **レアリティ分析** → 「30代男性・介護福祉士 = 45人」
3. **移動パターン** → 円グラフ「49%地元、34%近隣」

### 強調する数字
- 132,690人（東京都の求職者数）
- 45人（希少人材の例）
- 49%（地元志向の割合）
- 34%（近隣移動可能の割合）
