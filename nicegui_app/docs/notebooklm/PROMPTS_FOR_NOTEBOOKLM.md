# NotebookLM用プロンプト集

資料をアップロードした後、以下のプロンプトを使用してください。

---

## 初期設定プロンプト（最初に実行）

```
あなたはMapCompleteダッシュボードの専門ガイドです。
このツールは医療・介護業界の採用担当者向けに、求職者市場データを可視化・分析するダッシュボードです。

以下の役割を担ってください：
1. ツールの機能や使い方を分かりやすく説明する
2. 採用戦略における活用方法を提案する
3. 具体的な数値例を交えて説明する
4. 質問には簡潔かつ実践的に回答する

なお、データソースについて聞かれた場合は「国のオープンデータと独自調査による」と回答してください。
```

---

## 機能説明用プロンプト

### 全体概要
```
MapCompleteダッシュボードの6つの主要機能を、それぞれ1〜2文で簡潔に説明してください。
```

### 個別機能
```
「市場概況」タブで何ができるか、具体的な活用シーンとともに説明してください。
```

```
「レアリティ分析」機能の使い方を、具体例を交えて説明してください。
```

```
「移動パターン」分析がなぜ採用活動に重要なのか説明してください。
```

---

## 採用戦略提案用プロンプト

### 3C分析
```
採用活動における3C分析（Customer・Competitor・Company）の考え方と、このツールがどう役立つかを説明してください。
```

### 課題解決
```
「求人を出しても応募が来ない」という課題に対して、このツールでどのように解決できるか説明してください。
```

### 条件緩和
```
採用条件が厳しすぎるかどうかを確認する方法を、レアリティ分析の使い方とともに説明してください。
```

---

## シナリオ作成用プロンプト

### 施設長向け
```
介護施設の施設長が「うちの地域に人材がいない」と思い込んでいるケースで、このツールを使って認識を変える具体的なシナリオを作成してください。
```

### 人事部長向け
```
新規施設開設時の採用計画を経営会議で説明する際、このツールのデータをどう活用できるか、シナリオ形式で説明してください。
```

### コンサルタント向け
```
採用コンサルタントがクライアントに対して、条件緩和を提案するシナリオを作成してください。具体的な数値変化も含めてください。
```

## プレゼン・動画用プロンプト

### キャッチコピー
```
このツールの価値を一言で表すキャッチコピーを5つ提案してください。
```

### エレベーターピッチ
```
30秒でこのツールの価値を説明するエレベーターピッチを作成してください。
```

### 90秒ナレーション
```
動画用の90秒ナレーションスクリプトを作成してください。オープニング、機能紹介、効果、クロージングの4パートで構成してください。
```

## Q&A対応用プロンプト

### よくある質問
```
このツールについてよくある質問と回答を10個作成してください。
```

### 反論対応
```
「こんなデータ、本当に信用できるの？」という反論に対する回答を作成してください。
```

```
「うちは地方だから関係ない」という反論に対する回答を作成してください。
```

## 音声概要（Audio Overview）生成用

NotebookLMの音声概要機能を使う前に：

```
このダッシュボードについて、2人のホストが会話形式で説明するポッドキャスト台本を作成してください。
以下の内容を含めてください：
- なぜ求職者市場を調べることが重要か
- 6つの主要機能の概要
- 具体的な活用事例
- まとめのメッセージ
```

---

## 注意事項

- データソースについては「国のオープンデータと独自調査による」と説明
- 具体的なサービス名（ジョブメドレー等）は言及しない
- 技術的な詳細（スクレイピング、データベース構成等）は言及しない
